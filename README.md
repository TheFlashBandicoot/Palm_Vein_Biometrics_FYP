# Palm_Vein_Biometrics_FYP
Python code for an FYP on Palm Vein Detection using NIR imaging for Biometric Identification
--------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------

The i2p.py file is the application-layer communications protocol and contains the I2P client and Server Classes.
  This file is imported by the other Python scripts.
----------------------------------------------------------------------------------------------------------------------  
The vein_imager.py file is the final Python script which runs on the Raspberry Pi IoT device to capture images.
-------------------------------------------------------------------------------------------------------------------------------
The vein_processor.py file is the final Python script which runs on the laptop/server to receive and process the images. 
  This file is the final image processing solution.
  
 ------------------------------------------------------------------------------------------------------------------------------------------------
The image_receiver.py file was used to test all the various image processing techniques, and to generate images of each step for the report.
